## The Movement Keys :
     - Yellow Ship :
        - W key - UP
        - A key - LEFT
        - S key - DOWN
        - D key - RIGHT
     - Red Ship :
        - UP Arrow key - UP
        - LEFT Arrow key - LEFT
        - DOWN Arrow key - DOWN
        - RIGHT Aroow key - RIGHT
       
## The Fire keys :
     - Yellow Ship :
        - LCTRL - to fire the bullets 
     - Red Ship :
        - RCTRL - to fire the bullets
